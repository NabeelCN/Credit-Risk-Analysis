Credit Risk Analysis
Project Overview

This project focuses on predicting credit risk using Machine Learning classification techniques. The objective is to analyze customer financial data and classify whether an applicant is likely to be a low-risk or high-risk borrower. This helps financial institutions make better lending decisions and minimize the risk of loan defaults.

Objectives
Perform data cleaning and preprocessing
Conduct Exploratory Data Analysis (EDA)
Analyze factors affecting credit risk
Build and evaluate classification models
Predict whether a customer is creditworthy or risky
